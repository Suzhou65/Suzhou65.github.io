# 進攻方
**IQ 可以找到有電子裝置的東西**  
- 絆雷（Kapkan）  
- 電箱（Bandit）  
- 邪眼（Maestro）  
- 駭蟲（Mozzie）  
- 電擊盾（Clash   
- 蠱地雷（劉醒）  
- 干擾器（Mute）  
- 電極爪（Kaid）  
- 替身使者（Alibi）  
- 震撼詭雷（Ela）  
- 影像干擾器（Vigil）  
- 主動防禦系統（Jäger）  
- 智慧防攻入眼鏡（Warden）  
- 手持心跳偵測器（Paluse）  
- 妖怪無人機與遙控器（Echo）  
- 遙控炸藥（Nitro Cells, C4）  
- 監視攝影機  
- 防彈監視器  
- **IQ 看不到 Frost 的捕獸夾等純物理的陷阱，要特別注意**

**Maverick 的 Suri 裝置重點**  
- 可以無聲燒開強化的牆面與可破壞牆面、艙口  
- 可以燒毀邪眼（Maestro）跟防彈攝影機  
- 可以無視電箱（Bandit）的直接效果燒穿牆面，再破壞電箱  

**Blitz 的運用**  
- Blitz 的閃盾是用無線遙控的，會被干擾器（Mute）影響無法觸發  
- 只有**「視線正對」** Blitz 才會被閃瞎狗眼  
- Blitz 閃完之後不要用腰射，抓住被閃瞎狗眼的苦主那幾秒鐘開鏡用瞄的  
- 近身攻擊、震撼詭雷（Ela）、妖怪無人機（Echo）可以讓 Blitz 露出破綻  

**Montagne 的運用**  
- 近身攻擊、震撼詭雷（Ela）、妖怪無人機（Echo）可以強制讓 Montagne 收盾露出破綻  
- 在確保防守方沒有 C4 跟手榴彈的前提下，Montagne 可以在炸彈局用卡牆角的方式逼迫對方陷入僵局  
 
## 防守方
**Bandit 的電箱**  
- 不要一開始就把全部的電箱都用掉，留一個在身邊去仔細聽進攻方的行動  
- 進攻方的貼片炸藥可以再貼上後引爆前直接用電箱破壞  
- Ash 的爆破榴彈、Fuze 的人質救援裝置、Hibana 的花火、Thermite 的熱切炸藥在**觸發後就無法用電箱破壞**  

**Jäger 的 ADS 不能對應**  
- Capitão 的十字弓  
- Hibana 的花火  
- Ying 的燭光**（已經觸發的霰射狀態）**  
- Twitch 的小電車可以把 ADS 擊毀

**Frost 的迎賓踏墊**  
- Frost 的捕獸夾是純物理的陷阱，並不會被 IQ 發現  
- Thatcher 的 EMP 手榴彈並不能破壞捕獸夾  
- 推薦把捕獸夾放在鐵梯上來的地方，有奇效

**Clash 的防彈盾**  
- 一次電擊只能扣三點 HP，傷害不顯著  
- 電盾可以防彈、並對闊刀地雷跟手榴彈都有防禦力  
- 近身攻擊以及震盪榴彈（Zofia）可以強制讓 Clash 的盾牌出現破綻，Kali 的狙擊槍也可以  
- Clash 並不能像 Blitz 或是 Montagne 那樣可以持盾射擊或是使用盾牌近戰攻擊
